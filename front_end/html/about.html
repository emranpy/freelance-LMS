About CodeBuddy

CodeBuddy was created by the Piccolo lab. Our goal is to enable people around the world to more effectively learn how to write code and analyze data. With this Web-based tool, instructors create programming and data-analysis exercises that are organized into courses, assignments, and problems. For each problem, learners are presented with a description of the problem, as well as the expected output. The learners must then write code to match the expected output. Depending on the problem, these outputs can be either text-based or graphics-based.

CodeBuddy provides simple yet innovative functionality, including the following:

  1. Easy navigation among courses, assignments, and problems.
  2. Advanced code editing using Ace.
  3. The ability to create and edit problems via a graphical user interface. Problem descriptions are written using Markdown syntax, thus providing simple yet flexible formatting.
  4. An extensible design that supports the ability to create coding problems for virtually any programming language (currently, Python, R, and shell scripting are supported).
  5. Interactive feedback when learners submit code, including a panel that highlights differences between the learners' output and the expected output.
  6. The ability for learners to view the instructor's solution after successfully completing a given problem.
  7. Reports that indicate which courses, assignments, and problems have been attempted and/or completed by each student.

CodeBuddy is open source under the MIT license. We welcome bug reports and pull requests, but we encourage you to consult with us before attempting to add new features (we may already be working on them).
